인공지능 > 머신러닝 > 딥러닝 
1. 머신러닝 : 데이터에서 패턴을 학습 예측, 기계학습
    지도학습(정답 있는), 비지도학습(정답이 없는, 예측), 강화학습
2. 딥러닝 : 인공신경망, 이미지, 소리, 영상 -> 비정형데이터
    CNN(이미지), RNN(시퀀스), OpenCV, 트랜스포머(대규모 텍스트 데이터를 학습)...
3. LLM : 거대언어모델, 파라미터가 수십 ~ 수천억 -> 연산이 많이 들어감 -> 병렬처리 -> GPU -> 일반인이 할 수 없음
    GPT, BERT, 라마
    챗GPT(2022년) -> 생성형 AI
4. 멀티모달 : 텍스트 + 이미지 + 소리 + 영상 동시에 처리하는 LLM
    GPT-4, Gemini, Claude3
5. LAG -> Langchain(랭체인) : LLM 기능들을 모듈로 묶어 연결해서 앱처럼 만들수 있는 파이썬 프레임워크
6. LangGraph(랭그래프) -> 랭체인을 그래프구조로 확장
    워크플로우 자동화 프레임워크(쉽게해줌)
7. Agent(에이전트) -> 비서, LLM이 도구를 선택하고 실행을 판단 : 스스로 일처리하는 시스템
8. MCP -> 에이전트가 협업하는 프로토콜(통일된 규칙)
9. A2A -> 구글에서 만듦, 자연어로 소통
옵션. 바이브 코딩 ex)커서, 구글 CLI, 윈드서프, 레플잇, 코파일럿
